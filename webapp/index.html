What is this "sandbox" for?

This homepage is a demonstration of a POC R package which tries to filter "malicious" calls in R expressions based on a blacklist. This means if you know nothing of R and/or not interested in using R in a shared environment or in a web application, you might have better stuffs to do instead of lruking here :)

For more details please check out my Github repository!

The goal of this little environment is to let you test your R and other skills:

  • try to read the system-wide readable /sandbox/secret file,
  • try to write something in the system-wide writable /sandbox/hello file,
  • or simply try to figure out the root password on the machine :)

Some help: you may find some server-specific information on RApache info page.

If you would succeed in any above tasks, please give me some feedback in mail or on GitHub!

For monitoring purposes your R commands and your IP address besides a timestamp is logged.
